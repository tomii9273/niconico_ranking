# niconico_ranking

ニコニコ動画の累計 (全期間合計) の再生数などのランキングとその履歴を公開します。  
https://tomii9273.github.io/niconico_ranking/

## プルリクエストについて

機能・コードに大幅な変更・拡張を加えるプルリクエストは、マージしない場合があります。
